# Stage 1 - TS 語法入門

這一階段只做三件事：看懂型別、把函式型別補完整、開始習慣不可變資料。

## Lessons

- `S1-01` `01_basic_types.ts` - 基本型別與變數
- `S1-02` `02_functions.ts` - 函式、可選參數、Union
- `S1-03` `03_objects_arrays.ts` - 物件、陣列、不可變資料

## Recommended flow

1. `pnpm run learn:start -- S1-01`
2. 修改對應的 `.ts` 檔
3. `pnpm run learn:check -- S1-01`
4. 準備給 AI review 時，直接說「請 review S1-01」或先跑 `pnpm run learn:review -- S1-01`
