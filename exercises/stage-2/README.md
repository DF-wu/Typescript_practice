# Stage 2 - TS 型別核心能力

Stage 2 不是再背語法，而是開始把 TypeScript 當成設計工具來用。

## Lessons

- `S2-01` `01_narrowing.ts` - Narrowing 與 Type Guard
- `S2-02` `02_null_safety.ts` - Null Safety 與 Optional Chaining
- `S2-03` `03_utility_types.ts` - Utility Types
- `S2-04` `04_generic_repository.ts` - Generic Repository

## Recommended flow

1. `pnpm run learn:start -- S2-01`
2. 完成 TODO，先把輸出跑通
3. `pnpm run learn:check -- S2-01`
4. 需要我 review 時，直接報 lesson ID 即可
