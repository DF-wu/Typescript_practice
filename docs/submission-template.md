# Submission Template (Deprecated)

這份模板保留給你在**非常想手動整理一次內容**時使用。

現在的預設流程已經改成：

1. `pnpm run learn:check -- <lesson-id>`
2. 直接說：`請 review <lesson-id>`

如果你想把程式碼與輸出自動存成 markdown，請用：

```bash
pnpm run learn:review -- <lesson-id>
```
